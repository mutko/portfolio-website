Works

Below are listed few recent front-end projects.

Caffeine Rush

Caffeine Rush is a app designed to help people find good coffee within walking distance. It displays coffee shops near user's current location. Integral part of app is rich Foursquare api. App is built completely in model-view-controller architecture.

  • JavaSript
  • Foursquare API
  • OpenStreetMap API
  • Geolocation API
  • MVC pattern
  • HTML & SCSS
  • Webpack & Babel
See Project on Github

VueJS CRUD App

Basic CRUD App built with Vue.js and Vuex. Built as SPA using Vue Router. App have two main parts, classic landing app page and crud part which utilizes external api for users and messages data. After user register and login he manage list of users and messages.

  • VueJS
  • Vuex
  • Rest API
  • Bootstrap
  • HTML & SCSS
See Project on Github

Varex Welding Company

Wheater App
